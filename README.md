# MAS-Submod-Firefox
 MAS Submod for reactions in firefox. Helpful for wayland users

## Installation
 1) Install this plugin from https://addons.mozilla.org/ru/firefox/addon/mas-submod-plugin-for-firefox/
 2) Copy Firefox_Submod in game/Submod folder
